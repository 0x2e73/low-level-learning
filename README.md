## Why This Exists

I'm 22, working as a developer, but I realized I was building everything on abstractions I didn't truly understand. APIs, frameworks, libraries - all black boxes.

I want to understand the **full stack**, from silicon to software:
- How does a CPU actually execute instructions?
- Why are GPUs so much faster for certain tasks?
- How do frameworks like tinygrad generate optimized code?
- What happens when I write `a = b + c` at the hardware level?

This repo documents my journey learning low-level computing, systems architecture, and GPU programming. Raw notes, messy code, and honest confusions included.

**Not trying to become an expert overnight. Just learning consistently, 1-2h per day.**
